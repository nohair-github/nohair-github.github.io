No Hair: Blackhats banlist

Here are links to a blackhats banlist which is undated daily from output of the local reverse proxy running in front of three locally hosted web sites. The flagged IP addresses have tried to exploit wordpress, php, and other application which do not run on this server, or have tried to run various shell scripts.

The IP addresses may be spoofed but many are found on other published banlists such as Spamhaus. The list is sorted, duplicates are removed, and it can be used with pf or other firewalls. With a bit of editing, it is also useful for unbound-adblock, unwind-adblock, and various Pi-hole installations.

Download link: blacklists.bak

Hope you find it useful.
